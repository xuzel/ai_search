## ⚡ 常用命令速查

```bash
# 快速开始
python -m src.main ask "你的查询" --auto

# 使用特定模式
python -m src.main search "搜索查询"          # 研究模式
python -m src.main solve "计算问题"          # 代码执行
python -m src.main chat                      # 交互式聊天

# 显示系统信息
python -m src.main info

# 查看帮助
python -m src.main ask --help
```
